How to get Github Avatar link

1/09/2014

  1. Go to
    2.   
    https://api.github.com/users/<github-username>
  2. Copy URL variable related to avatar_url
  3. Include this link in img src
  4. And you are done

Show password of hosted network in Windows 7

  1. Goto Start > Run
  2. Type cmd
  3. In command line, type
    4.   
    netsh wlan show hostednetwork setting=security
